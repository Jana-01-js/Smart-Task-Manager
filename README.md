# Smart Task Manager

A responsive web application built with HTML, CSS, and JavaScript to manage daily tasks efficiently.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search tasks instantly
- Persistent storage using Local Storage
- Responsive design for mobile and desktop

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
